# Frontend Guide

## Cai dat

```bash
cd frontend
npm install
```

## Cau hinh API

Tao file `.env` tu `.env.example`:

```bash
VITE_API_BASE=http://127.0.0.1:8000
```

## Chay local

```bash
cd frontend
npm run dev
```

Frontend mac dinh:

- `http://127.0.0.1:5173`

## Build production

```bash
cd frontend
npm run build
```

## Ghi chu

- frontend hien tai da noi truc tiep voi backend FastAPI
- card ket qua da dung:
  - local unit avatars
  - local trait icons
  - enriched search payload
- UI duoc thiet ke theo huong hien dai hon, de mo rong thanh website thuc te
